# Apuntes y recursos — UAGRM Semestre I (2026)

Repositorio personal para almacenar apuntes, lecturas, códigos y proyectos de las materias del primer semestre 2026 en la UAGRM.

Este repositorio está organizado por asignaturas y contiene materiales de estudio (apuntes en Markdown), lecturas, ejemplos de código y carpetas de proyecto cuando corresponda.

## Estructura principal

| Carpeta | Contenido | Comentario |
|--------:|----------|-----------|
| `INF412-Sistemas-Informacion-II/` | Apuntes, lecturas, código y proyecto del curso | Ej.: `Apuntes/2026-02-03-Primera-Clase.md` |
| `INF413-Sistemas-Operativos-II/` | Lecturas y apuntes sobre SO | Ej.: `Lectura/01-Almacenamiento-Secundario.md` |
| `INF423-Redes-II/` | Lecturas y material de redes | Carpeta `Lectura/` para resúmenes y apuntes |
| `INF432-Sistemas-Soporte-Decision/` | Lecturas y apuntes de soporte a la decisión | Contiene resúmenes y material adicional |

> Nota: cada materia puede contener subcarpetas como `Lectura/`, `Apuntes/`, `Codigo/` o `Proyecto/`.

## Cómo usar este repositorio

- Buscar por materia y entrar en la subcarpeta correspondiente (`Apuntes`, `Lectura`, `Codigo`, `Proyecto`).
- Los apuntes están en formato Markdown para facilitar lectura y edición en VS Code.
- Se incluyen diagramas Mermaid y fórmulas en LaTeX cuando es útil para visualización o cálculo.

## Ejemplos de archivos útiles

- `INF412-Sistemas-Informacion-II/Apuntes/Primera-Clase.md` — plantilla de apuntes y recomendaciones para proyectos.
- `INF413-Sistemas-Operativos-II/Lectura/01-Almacenamiento-Secundario.md` — apuntes sobre dispositivos de almacenamiento.
- `INF413-Sistemas-Operativos-II/Lectura/02-Archivos.md` — apuntes sobre archivos y sistemas de archivos.
